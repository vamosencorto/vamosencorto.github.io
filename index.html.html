Herramientas

Como usar las herramientas

Instrumentos y software esenciales para ingenieria y tecnologia. Aprende a usarlos correctamente.

MEDICION
Multimetro Digital
Mide voltaje, corriente, resistencia y continuidad en circuitos electronicos e instalaciones electricas.
  • Selecciona la magnitud a medir (VDC, VAC, A, Ohm)
  • Elige el rango adecuado o usa autorange
  • Conecta cables: rojo a + (V/Ohm) y negro a COM
  • Para corriente, inserta en serie con el circuito
  • Lee el resultado en la pantalla digital
ANALISIS
Osciloscopio
Visualiza senales electricas en el tiempo. Mide frecuencia, amplitud, desfase y forma de onda.
  • Conecta la sonda al canal CH1 y la pinza a GND
  • Ajusta la escala de tiempo (Time/Div)
  • Ajusta la escala de voltaje (Volts/Div)
  • Usa el trigger para estabilizar la senal
  • Mide frecuencia: f = 1 / periodo
SOFTWARE
Arduino IDE
Entorno de programacion para microcontroladores Arduino y ESP32. Basado en C/C++.
  • Descarga Arduino IDE en arduino.cc
  • Instala el driver del microcontrolador
  • Selecciona la placa en Herramientas > Placa
  • Escribe codigo en setup() y loop()
  • Compila y sube con el boton Upload
PROGRAMACION
Python + VS Code
Entorno de desarrollo profesional para Python, ideal para ingenieria, datos y automatizacion.
  • Instala Python en python.org y VS Code en code.visualstudio.com
  • Instala la extension Python en VS Code
  • Crea un entorno virtual: python -m venv env
  • Instala librerias: pip install numpy matplotlib
  • Ejecuta con F5 o el boton Run
PLC
TIA Portal (Siemens)
Software profesional para programar PLCs Siemens S7-1200 y S7-1500 con Ladder, FBD y ST.
  • Crea un nuevo proyecto y agrega el PLC S7-1200
  • Configura la direccion IP del PLC
  • Programa en Ladder o Bloques de Funcion
  • Compila con Ctrl+B y verifica errores
  • Descarga al PLC con el boton Download
DISENIO PCB
KiCad - Disenio PCB
Software gratuito y profesional para disenar esquematicos y tarjetas de circuito impreso PCB.
  • Crea el esquematico en Eeschema
  • Asigna footprints a cada componente
  • Genera netlist y abre PCBnew
  • Coloca componentes y traza pistas de cobre
  • Exporta archivos Gerber para fabricacion
01 — ELECTRONICA
Electronica
Control del flujo de electrones en semiconductores y circuitos para procesar informacion y controlar sistemas. Desde componentes basicos hasta microcontroladores modernos.

Ley de Ohm y conceptos basicos

La electronica estudia el control del flujo de electrones en semiconductores. Los conceptos fundamentales son carga, corriente (I), tension (V) y resistencia (R). Los materiales se clasifican en conductores, semiconductores y aislantes.

LEY DE OHM
V = I × R
POTENCIA ELECTRICA
P = V × I = I² × R = V²/R
EJEMPLO PRACTICO

Si tienes un LED con R=220Ω conectado a 5V: I = V/R = 5/220 = 22.7mA — corriente segura para el LED.

Materiales semiconductores

El silicio y germanio son los semiconductores mas usados. Su conductividad se controla mediante dopaje con impurezas tipo P (huecos) y tipo N (electrones). La union P-N forma el diodo.

Diodos y Rectificacion

El diodo permite el paso de corriente en una sola direccion. La tension umbral es ~0.7V para silicio. Tipos: rectificador, Zener, LED, Schottky, fotodiodo y varactor.

MODELO DIODO IDEAL
VD ≈ 0.7V (silicio)
PUENTE RECTIFICADOR
VDC = 0.9 × VAC
APLICACION REAL

Un puente rectificador de 4 diodos convierte 12VAC del transformador en ~10.8VDC para alimentar circuitos electronicos.

Diodo Zener

El diodo Zener conduce en inversa a un voltaje fijo (Vz). Es fundamental para regulacion de tension de referencia y proteccion contra sobretensiones en fuentes de alimentacion.

Transistores BJT y MOSFET

El BJT controla corriente de colector mediante corriente de base. El MOSFET controla mediante voltaje de puerta, con muy alta impedancia de entrada y sin consumo de corriente de control.

TRANSISTOR BJT
IC = β × IB
GANANCIA TIPICA
β (hFE) = 100 a 500
EJEMPLO PRACTICO

Con β=200 y IB=50µA, el transistor conduce IC = 200×50µA = 10mA — suficiente para activar un rele de 5V.

MOSFET de potencia

Los MOSFET de potencia (IRF540, IRLZ44N) conmutan amperios con solo voltaje de puerta. Son la base de variadores de velocidad, cargadores y convertidores switching modernos.

Amplificadores Operacionales

El Op-Amp es un amplificador diferencial de alta ganancia. Con realimentacion negativa se disenan: amplificador inversor, no inversor, sumador, integrador y diferenciador.

AMPLIFICADOR INVERSOR
Vout = -(Rf/Rin) × Vin
AMPLIFICADOR NO INVERSOR
Vout = (1 + Rf/R1) × Vin
EJEMPLO PRACTICO

Con Rf=10kΩ y Rin=1kΩ, ganancia del inversor = -10. Si Vin=0.5V entonces Vout=-5V.

Filtros activos

Los filtros Butterworth y Chebyshev usan Op-Amps para filtrado de senal. Se usan en instrumentacion medica, audio profesional y acondicionamiento de sensores industriales.

Electronica Digital

Trabaja con estados logicos 1 y 0. Compuertas AND, OR, NOT, NAND, NOR, XOR son bloques fundamentales. El algebra de Boole describe la logica combinacional.

LEYES DE DE MORGAN
¬(A·B) = ¬A + ¬B
COMPUERTA NAND UNIVERSAL
NAND(A,A) = NOT(A)
APLICACION REAL

Un flip-flop D almacena un bit de informacion. 8 flip-flops forman un registro de 8 bits. Millones de ellos forman la RAM de tu computadora.

Sistemas numericos

Binario (base 2), octal (base 8) y hexadecimal (base 16) son sistemas usados en programacion. Ejemplo: 0xFF = 255 decimal = 11111111 binario.

Microcontroladores

Integran procesador, memoria Flash, RAM y perifericos (GPIO, UART, SPI, I2C, ADC, PWM). Arduino, PIC, STM32 y ESP32 son los mas populares en educacion e industria.

RESOLUCION ADC
Vmin = Vref / 2N
EJEMPLO ADC 10-BIT
Vmin = 5V / 1024 = 4.88mV
CODIGO ARDUINO

int valor = analogRead(A0); — Lee el ADC del pin A0 y devuelve un valor entre 0 y 1023 proporcional al voltaje (0 a 5V).

ESP32 y WiFi

El ESP32 integra WiFi y Bluetooth con un procesador dual-core de 240MHz. Ideal para proyectos IoT, domótica y sistemas de monitoreo remoto inalambrico de bajo costo.

07
Sensores y Transductores
Temperatura (PT100, NTC, LM35), presion (MEMS), humedad (capacitivo), aceleracion (IMU), luz (fotodiodo). El acondicionamiento adapta la senal al ADC.
Tipos principales
Clasificacion de sensores
Activos (generan senal: termopar, piezoelectrico) y pasivos (modifican parametro: RTD, LDR, strain gauge). La eleccion depende de precision, rango y condiciones ambientales.
08
Fuentes de Alimentacion
Reguladores lineales (7805): simples pero ineficientes. Fuentes switching (SMPS): 85-95% eficiencia. Convertidores Buck (reductor), Boost (elevador), Flyback (aislado).
Formula clave
Eficiencia de conversion
η = P_salida / P_entrada × 100%. Un regulador lineal de 12V a 5V con 1A disipa 7W en calor. Un Buck switching disipa solo ~0.7W con 95% eficiencia.
09
Disenio de PCB
Proceso: esquematico en EDA → asignar footprints → layout → DRC → fabricacion. Herramientas: KiCad (gratis), Altium (profesional). Los planos GND reducen EMI.
Reglas basicas
Buenas practicas de layout
Pistas de potencia: minimo 1mm. Senales: 0.2mm. Condensadores de desacoplo de 100nF junto a cada CI. Via de 0.8mm para conexion entre capas. GND en capa completa.
10
Compatibilidad Electromagnetica EMC
Asegura funcionamiento correcto sin interferencias. Tecnicas: planos de tierra, condensadores de desacoplo, filtros, blindaje metalico y trazado cuidadoso de pistas de alta frecuencia.
Normativa
Certificaciones requeridas
Marcado CE requiere conformidad EMC en Europa (IEC 61000). FCC Part 15 en USA. Los ensayos se realizan en camara anecoica. Disenar con conciencia EMC desde el inicio es clave.
02 — ROBOTICA
Robotica
Ciencia del diseno, construccion y operacion de robots. Combina mecanica, electronica, informatica e inteligencia artificial para crear sistemas autonomos.

Introduccion a la Robotica

La robotica es la ciencia dedicada al diseno, construccion y operacion de robots. Se clasifican en industriales, moviles, humanoides y de servicio. Combina mecanica, electronica e IA.

GRADOS DE LIBERTAD (DOF)
DOF = 6 (traslacion X,Y,Z + rotacion Roll,Pitch,Yaw)
EJEMPLO

Un brazo robotico de 6 DOF puede alcanzar cualquier posicion y orientacion en su espacio de trabajo, igual que el brazo humano.

Clasificacion de robots

Industriales: soldadura y ensamblaje. Moviles: AGVs y drones. Cobots: colaboran con humanos. Medicos: cirugia de precision. Cada tipo tiene arquitectura mecanica y sistema de control especifico.

Cinematica de Robots

Estudia el movimiento sin considerar fuerzas. La cinematica directa calcula la posicion del efector dando los angulos articulares. La inversa calcula los angulos dada la posicion deseada.

JACOBIANO DE VELOCIDADES
ẋ = J(q) × q̇
PARAMETROS DENAVIT-HARTENBERG
T = Rot(z,θ) × Trans(z,d) × Trans(x,a) × Rot(x,α)
APLICACION

Para un robot de 2 DOF plano: x = L1·cos(θ1) + L2·cos(θ1+θ2), y = L1·sin(θ1) + L2·sin(θ1+θ2).

03
Dinamica de Robots
Relaciona fuerzas, pares y movimientos. Ecuaciones de Lagrange o Newton-Euler. Incluye matrices de inercia, terminos de Coriolis, centrifugos y gravitacionales para control preciso.
Formula
Ecuacion de movimiento
M(q)·q̈ + C(q,q̇)·q̇ + G(q) = τ — Donde M es inercia, C son terminos de Coriolis, G es gravedad y τ son los pares aplicados por los actuadores en cada articulacion.
04
Sensores en Robotica
Propioceptivos: encoders, acelerometros, giroscopios. Exteroceptivos: camaras, LiDAR, ultrasonido, sensores de fuerza. La fusion sensorial combina datos para mayor robustez autonoma.
LiDAR
Sensor de distancia por laser
Mide distancias mediante pulsos laser. Resolucion angular de 0.1 grados. Alcance de 10 a 200m. Esencial para navegacion autonoma de vehiculos y robots moviles en interiores y exteriores.

Control PID en Robotica

El PID regula el comportamiento dinamico del robot. Es el controlador mas usado por su simplicidad y efectividad para seguimiento de trayectorias y control de posicion articular.

CONTROLADOR PID
u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·(de/dt)
ERROR
e(t) = referencia(t) - salida(t)
SINTONIZACION

Metodo Ziegler-Nichols: aumenta Kp hasta oscilacion → mide Ku y Tu → Kp=0.6Ku, Ti=Tu/2, Td=Tu/8.

06
ROS - Robot Operating System
Framework open-source para desarrollo robotico. Arquitectura de nodos que se comunican via topics y servicios. Incluye Rviz, Gazebo y rosbag. ROS2 soporta tiempo real con DDS.
Comando basico
Publicar y suscribirse
ros2 topic pub /cmd_vel geometry_msgs/Twist — Publica velocidades al robot. ros2 topic echo /scan — Muestra datos del LiDAR. La arquitectura modular facilita reuso de paquetes existentes.
07
Robots Moviles y SLAM
SLAM construye mapa mientras se localiza en el. Metodos: EKF-SLAM, FastSLAM, Graph-SLAM, ORB-SLAM visual. Algoritmos A* y RRT planifican rutas eficientes en tiempo real.
Aplicaciones
Donde se usa SLAM
Aspiradoras roboticas (Roomba), vehiculos autonomos, drones de inspeccion, robots de almacen (Amazon), exploracion planetaria (Mars Rover). Opera sin GPS en entornos desconocidos.
08
Robots Industriales y Cobots
Brazos de 6 DOF (KUKA, ABB, FANUC) dominan manufactura. Cobots (Universal Robots, KUKA iiwa) trabajan con humanos con seguridad integrada. ISO 10218 e ISO/TS 15066 regulan seguridad.
ROI tipico
Retorno de inversion
La automatizacion robotica logra ROI en 1-3 anos en produccion masiva. Disponibilidad 24/7, reduccion de rechazos hasta 99%, consistencia de calidad superior a manufactura manual.
09
IA en Robotica
Aprendizaje por refuerzo (RL) entrena robots mediante prueba-error. CNN para vision. LLMs como planificadores de lenguaje natural. LfD: aprendizaje por demostracion del experto humano.
10
Robotica Medica
Da Vinci: cirugia minimamente invasiva. Exoesqueletos ReWalk: paralisis. Lokomat: rehabilitacion automatizada. Nanorobots: cirugia celular futura. Normas IEC 60601 y regulaciones FDA.
03 — ELECTRICIDAD
Electricidad
Instalaciones electricas, maquinas electricas, sistemas de distribucion y protecciones. Desde fundamentos de circuitos hasta sistemas trifasicos industriales.

Circuitos de Corriente Alterna

La CA es la forma de electricidad usada en redes de distribucion. La tension y corriente varian sinusoidalmente con el tiempo. La impedancia combina resistencia, inductancia y capacitancia.

IMPEDANCIA COMPLEJA
Z = R + j(XL - XC)
REACTANCIA INDUCTIVA
XL = 2π × f × L
EJEMPLO

A 60Hz, una bobina de 10mH tiene XL = 2π × 60 × 0.01 = 3.77Ω de reactancia inductiva.

Factor de potencia

El coseno del angulo de desfase entre tension y corriente. FP = P/S = cos(φ). Un FP bajo indica consumo ineficiente y requiere compensacion con condensadores para mejorar eficiencia.

02
Maquinas Electricas
Motores de induccion (los mas usados en industria), motores sincrono (generadores), motores CC (control preciso de velocidad). Generadores convierten energia mecanica en electrica.
Ley de Faraday
Principio de operacion
EMF = -N × dΦ/dt — La FEM inducida es proporcional al numero de espiras y la variacion del flujo magnetico. Este principio rige todos los transformadores y maquinas electricas rotativas.
03
Transformadores
Convierte tensiones CA manteniendo la potencia (con perdidas minimas). Relacion de transformacion: V1/V2 = N1/N2 = I2/I1. Eficiencia tipica: 95-99%. Usos: distribucion, aislamiento, adaptacion.
Formula clave
Relacion de transformacion
n = N1/N2 = V1/V2 — Si un transformador tiene 1000 espiras primarias y 100 secundarias, transforma 220V a 22V. La potencia S = V×I se conserva (despreciando perdidas del nucleo).
04
Instalaciones Electricas
Residenciales (NOM-001-SEDE), industriales y comerciales. Calculo de conductores por capacidad de corriente y caida de tension. Protecciones, puesta a tierra y canalizaciones segun norma.
Caida de tension
Calculo de conductores
ΔV = 2 × ρ × L × I / S — Donde ρ es resistividad del cobre (0.0172 Ω·mm²/m), L es longitud, I es corriente y S es seccion del conductor. Maximo permitido: 3% en instalaciones residenciales.

Motores Electricos

El motor de induccion trifasico es el motor mas usado en industria. Funciona por campo magnetico rotante. La velocidad sincrona depende de la frecuencia y el numero de polos.

VELOCIDAD SINCRONA
Ns = 120 × f / P (RPM)
DESLIZAMIENTO
s = (Ns - Nr) / Ns × 100%
EJEMPLO

Motor 4 polos a 60Hz: Ns = 120×60/4 = 1800 RPM. Con deslizamiento del 3%, velocidad real = 1746 RPM.

Potencia Electrica

La potencia en sistemas electricos se divide en activa (P), reactiva (Q) y aparente (S). El triangulo de potencias relaciona estas tres magnitudes a traves del factor de potencia.

TRIANGULO DE POTENCIAS
S² = P² + Q²
POTENCIA ACTIVA TRIFASICA
P = √3 × VL × IL × cos(φ)
EJEMPLO

Motor trifasico 220V, 10A, cos(φ)=0.85: P = √3 × 220 × 10 × 0.85 = 3.23 kW de potencia util.

07
Protecciones Electricas
Interruptores automaticos MCB protegen circuitos. Fusibles son proteccion simple. Diferenciales RCD detectan fugas desde 30mA. Reles de proteccion en media tension: sobrecorriente, diferencial, distancia.
08
Energias Renovables
Paneles fotovoltaicos: monocristalino, policristalino, CdTe. Aerogeneradores onshore y offshore. Hidroelectrica: turbinas Pelton, Francis, Kaplan. El costo solar bajo 90% en 10 anos.

Sistemas Trifasicos

Tres corrientes alternas desfasadas 120° entre si. Mayor eficiencia que monofasico para igual potencia. Conexiones Estrella (Y) y Triangulo (Δ) con diferentes relaciones tension-corriente.

CONEXION ESTRELLA
VL = √3 × Vf ≈ 1.732 × Vf
POTENCIA TRIFASICA
P = √3 × VL × IL × cos(φ)
EJEMPLO

Red trifasica 127V/fase en estrella: VL = √3 × 127 = 220V de tension de linea — la tension estandar en Mexico para instalaciones trifasicas.

10
Seguridad Electrica
Umbrales de corriente: 1mA percepcion, 10mA contraccion muscular, 100mA fibrilacion ventricular (fatal). Protecciones: aislamiento, tierra, diferenciales, tension de seguridad menor a 50VCA.
Normativa
Normas aplicables en Mexico
NOM-001-SEDE instalaciones electricas. NOM-029-STPS para trabajos en tension. EPP obligatorio: guantes dielectricos, casco, calzado y trajes resistentes al arco electrico segun nivel de riesgo.
04 — AUTOMATIZACION
Automatizacion Industrial
PLCs, SCADA, HMI, redes industriales y control de procesos. Desde la logica escalera hasta la Industria 4.0 con IoT y gemelos digitales.

PLCs - Controladores Logicos Programables

El PLC escanea ciclicamente entradas (sensores, botones), ejecuta el programa y actualiza salidas (actuadores, reles). Inventado en 1968 por Dick Morley para General Motors.

CICLO DE ESCANEO PLC
Leer Entradas → Ejecutar Programa → Actualizar Salidas
LENGUAJES IEC 61131-3
Ladder (LD), FBD, ST, SFC, IL
LADDER BASICO

Contacto Normalmente Abierto I0.0 en serie con bobina Q0.0: si se activa el sensor I0.0, se activa la salida Q0.0 (motor, valvula, etc.).

PLCs mas usados en Mexico

Siemens S7-1200/1500 (TIA Portal), Allen-Bradley CompactLogix (Studio 5000), Mitsubishi MELSEC, Omron NX/NJ. El S7-1200 es el mas ensenado en universidades tecnicas mexicanas.

SCADA
Supervision y Adquisicion de Datos
Monitorea procesos distribuidos geograficamente. Componentes: MTU, RTUs/PLCs, red de comunicacion y software. Paquetes: Wonderware InTouch, IGNITION, WinCC, FactoryTalk.
Ciberseguridad
SCADA y amenazas digitales
Stuxnet demostro la vulnerabilidad de sistemas industriales a ciberataques. Los SCADA modernos implementan segmentacion de redes, firewalls industriales y protocolo IEC 62443 para proteccion.
Sensores
Sensores Industriales 4-20mA
Temperatura: termopar K (-200 a 1350°C), PT100 (-200 a 600°C). Presion: piezoresistivo. Nivel: ultrasonico, radar. Caudal: Coriolis, magnetico. Estandar 4-20mA inmune a ruido electrico.
Calibracion
Por que es obligatoria
Un sensor mal calibrado genera productos defectuosos, danos a equipos o condiciones peligrosas. La calibracion periodica establece la relacion magnitud-senal con trazabilidad a patrones nacionales.
Actuadores
Tipos de actuadores industriales
Neumaticos: limpios y rapidos para manufactura discreta. Hidraulicos: fuerzas enormes para prensas. Servomotores: posicionamiento preciso. VFDs: control de velocidad de motores de induccion.
VFD
Variadores de frecuencia
Controlan velocidad del motor variando frecuencia (0 a 60/50Hz) y voltaje. Ahorro energetico del 30-50% en bombas y ventiladores. Marcas: Siemens SINAMICS, ABB ACS, Danfoss VLT.
Redes
Redes Industriales
PROFIBUS DP (Siemens, 12Mbps), DeviceNet (Allen-Bradley), Modbus RTU/TCP (universal), PROFINET (Ethernet industrial Siemens), EtherNet/IP (Rockwell). OPC-UA es estandar de Industria 4.0.
Modbus
Protocolo universal
Protocolo simple y gratuito. Modbus RTU (RS-485 serie) y Modbus TCP (Ethernet). Funciones: 03 Read Holding Registers, 06 Write Single Register. Compatible con casi todos los dispositivos industriales.

Control PID en Procesos Industriales

Mas del 95% de los lazos de control industrial usan PID. Mantiene variables de proceso (temperatura, presion, nivel, caudal, pH) en sus valores de referencia con minimo error.

CONTROLADOR PID
u(t) = Kp·e + Ki·∫e·dt + Kd·de/dt
ZIEGLER-NICHOLS (METODO CLASICO)
Kp=0.6Ku, Ti=Tu/2, Td=Tu/8
EJEMPLO PRACTICO

Control de temperatura de horno: SP=200°C, PV=180°C, e=20°C. El integral elimina el error en estado estacionario para mantener exactamente 200°C.

Neumatica
Sistemas Neumaticos e Hidraulicos
Neumatica: compresor + FRL (filtro, regulador, lubricador) + electrovalvulas 5/2 y 5/3 + cilindros. Hidraulica: fuerzas enormes para prensas y maquinaria pesada. FESTO y SMC dominan el mercado.
Circuito basico
Secuencia A+B+A-B-
Cilindro A avanza (A+), cilindro B avanza (B+), A retrocede (A-), B retrocede (B-). Se programa con relays o PLC mediante diagrama de secuencia. Presion tipica: 6-8 bar de trabajo.
Industria 4.0
Cuarta Revolucion Industrial
Integra IoT Industrial, big data, IA, gemelos digitales, manufactura aditiva, AR/VR, robotica colaborativa y ciberseguridad. Plataformas: Siemens MindSphere, PTC ThingWorx, AWS IoT.
Mantenimiento predictivo
IA para prediccion de fallos
Sensores IIoT monitorean vibracion, temperatura y corriente continuamente. Algoritmos de ML detectan anomalias horas antes del fallo. Reduce paradas no planificadas hasta 50% en plantas industriales.
05 — PROGRAMACION
Programacion
Desde algoritmos basicos hasta programacion orientada a objetos, bases de datos y control de versiones. Python, C++, Git y mas.

Fundamentos de Programacion

Variables, tipos de datos, operadores, estructuras de control (if, while, for), funciones y modularidad. Los algoritmos son secuencias finitas de pasos para resolver problemas eficientemente.

NOTACION BIG-O (COMPLEJIDAD)
O(1) < O(log n) < O(n) < O(n log n) < O(n²)
EJEMPLO PYTHON

def suma(a, b): return a + b — Funcion simple. Para n numeros: for i in range(n): total += lista[i] es O(n) lineal.

Paradigmas de programacion

Imperativo (instrucciones secuenciales), OOP (objetos y clases), funcional (funciones puras sin efectos laterales) y declarativo (que hacer, no como). Python soporta todos los paradigmas.

Estructuras
Estructuras de Datos
Array O(1) acceso. Lista enlazada O(1) insercion. Pila LIFO (push/pop). Cola FIFO (enqueue/dequeue). Arbol BST O(log n) busqueda. Hash table O(1) promedio. Grafo para redes y rutas.
Cuando usar cual
Seleccion de estructura
Acceso frecuente por indice → array. Inserciones/eliminaciones frecuentes → lista enlazada. Historial de operaciones → pila. Cola de tareas → cola. Busqueda rapida → hash table o BST.
OOP
Programacion Orientada a Objetos
Pilares: encapsulamiento (datos privados), herencia (reusar codigo), polimorfismo (misma interfaz diferente comportamiento), abstraccion (simplificar complejidad). Lenguajes: Java, C++, Python, C#.
SOLID
Principios de diseno
S: responsabilidad unica. O: abierto/cerrado. L: sustitucion Liskov. I: segregacion de interfaces. D: inversion de dependencias. Seguirlos produce codigo mantenible, testeable y extensible.
Algoritmos
Ordenamiento y Busqueda
Bubble Sort O(n²). Merge Sort O(n log n). Quick Sort O(n log n) promedio. Heap Sort O(n log n). Busqueda lineal O(n). Busqueda binaria O(log n) — requiere lista ordenada.
Grafos
Algoritmos de grafos
BFS (anchura): camino mas corto sin pesos. DFS (profundidad): detectar ciclos, componentes. Dijkstra: camino minimo con pesos positivos. A*: con heuristica para navegacion y videojuegos.

Python para Ingenieria

Lenguaje mas popular para ingenieria, ciencia de datos e IA. NumPy para computacion numerica, SciPy para algoritmos cientificos, Matplotlib para graficas, Pandas para datos, Scikit-learn para ML.

NUMPY - OPERACION VECTORIAL
import numpy as np; A = np.array([1,2,3])
EJEMPLO INGENIERIA

import numpy as np; t = np.linspace(0,1,1000); signal = np.sin(2*np.pi*60*t) — Genera 1 segundo de senal sinusoidal a 60Hz.

C y C++
Lenguajes de sistemas
C es rapido, portable y de bajo nivel. Punteros y gestion manual de memoria son su caracteristica distintiva. C++ anade OOP, templates y STL. Esenciales para embebidos, kernels y robotica.
Punteros en C
Concepto fundamental
int *p = &x; — p guarda la direccion de x. *p accede al valor. Los punteros permiten manipulacion directa de memoria, paso por referencia y estructuras dinamicas como listas enlazadas y arboles.
SQL
Bases de Datos Relacionales
SELECT nombre FROM empleados WHERE salario > 10000 ORDER BY nombre — Consulta basica SQL. MySQL, PostgreSQL, SQLite son los RDBMS mas usados. Las transacciones ACID garantizan integridad.
Normalizacion
Diseno de base de datos
1NF: elimina grupos repetidos. 2NF: elimina dependencias parciales. 3NF: elimina dependencias transitivas. Un buen diseno normalizado elimina redundancias y garantiza integridad referencial de datos.
Git
Control de Versiones
git init, clone, add, commit, push, pull, merge, branch, rebase. Las ramas permiten desarrollo paralelo. GitHub/GitLab alojan repositorios. Pull Requests facilitan revision de codigo en equipo.
Flujo basico
GitFlow simplificado
main: codigo en produccion. develop: integracion. feature/*: nuevas funcionalidades. hotfix/*: correcciones urgentes. Cada commit debe ser atomico con mensaje descriptivo del cambio realizado.
06 — MATEMATICAS
Matematicas
Algebra lineal, calculo diferencial e integral, ecuaciones diferenciales, Transformada de Laplace y Fourier. El lenguaje matematico de la ingenieria moderna.

Algebra Lineal

Vectores, matrices, espacios vectoriales y transformaciones lineales. Las matrices representan transformaciones y sistemas de ecuaciones. Los eigenvalores caracterizan transformaciones lineales.

DETERMINANTE 2x2
det(A) = ad - bc
EIGENVALORES
det(A - λI) = 0
APLICACION EN INGENIERIA

En machine learning: PCA usa eigenvalores para reducir dimensionalidad. En robotica: la matriz de rotacion R (3x3) describe la orientacion del efector en el espacio cartesiano.

Calculo Diferencial

Estudia el cambio instantaneo de funciones. La derivada mide la tasa de cambio. Aplicaciones: maximos y minimos, optimizacion, velocidad e aceleracion, recta tangente, aproximacion de Taylor.

DEFINICION DE DERIVADA
f'(x) = lim[h→0] (f(x+h) - f(x)) / h
REGLA DE LA CADENA
d/dx[f(g(x))] = f'(g(x)) · g'(x)
EJEMPLO ELECTRICO

Si v(t) = 5·sin(2πt), la corriente en un capacitor es i = C·dv/dt = C·5·2π·cos(2πt) = 10πC·cos(2πt).

Integral
Calculo Integral
Operacion inversa de la derivada. Mide areas, volumenes y acumulaciones. Teorema Fundamental conecta derivada e integral. Tecnicas: sustitucion, partes, fracciones parciales, trigonometrica.
Formula
Integral definida
∫[a,b] f(x)dx = F(b) - F(a) — Calcula el area bajo la curva. Aplicacion electrica: Q = ∫i(t)dt calcula la carga total en un capacitor. Trabajo: W = ∫F(x)dx con fuerza variable.

Ecuaciones Diferenciales

Relacionan funciones con sus derivadas, modelando fenomenos dinamicos. ODEs de primer orden: variables separables, lineales. Segundo orden: coeficientes constantes. Metodo de Laplace simplifica la solucion.

EDO LINEAL 1er ORDEN
dy/dx + P(x)·y = Q(x)
CIRCUITO RC (RESPUESTA)
v(t) = V₀·(1 - e^(-t/RC))
APLICACION RC

Un capacitor de 100µF cargando a traves de 1kΩ: τ = RC = 0.1s. Al tiempo t=τ, el capacitor tiene el 63.2% de la tension final de alimentacion.

Transformada de Laplace

Convierte funciones del tiempo f(t) en funciones de la variable compleja s. Transforma ecuaciones diferenciales en algebraicas. Fundamental para diseno de sistemas de control y filtros.

DEFINICION
L{f(t)} = F(s) = ∫[0,∞] f(t)·e^(-st) dt
PARES COMUNES
u(t) → 1/s | e^(-at) → 1/(s+a) | sin(ωt) → ω/(s²+ω²)
FUNCION DE TRANSFERENCIA

H(s) = Y(s)/U(s) — Caracteriza sistemas lineales. Un filtro RC pasa-bajas: H(s) = 1/(1+sRC). La frecuencia de corte es ωc = 1/RC rad/s.

Boole
Algebra de Boole
Variables 0 y 1. Operaciones AND (·), OR (+), NOT (¬). Leyes De Morgan: ¬(A·B) = ¬A+¬B. Mapas de Karnaugh minimizan expresiones booleanas. Base matematica de todos los computadores digitales.
Ejemplo
Simplificacion con Karnaugh
Expresion: F = ABC + AB'C + A'BC se simplifica a F = AC + BC agrupando minterminos adyacentes en el mapa de Karnaugh de 3 variables, reduciendo de 3 a 2 productos necesarios en el circuito.
Vectorial
Calculo Vectorial
Gradiente ∇f: direccion de maximo incremento. Divergencia ∇·F: fuentes de un campo vectorial. Rotacional ∇×F: rotacion del campo. Teoremas de Green, Stokes y Gauss-Ostrogradski.
Electromagnetismo
Ecuaciones de Maxwell
∇·E = ρ/ε₀ (Ley de Gauss). ∇×B = µ₀J + µ₀ε₀∂E/∂t (Ampere-Maxwell). Estas 4 ecuaciones describen todos los fenomenos electromagneticos y son la base de las telecomunicaciones modernas.
Estadistica
Probabilidad y Estadistica
Distribucion Normal (Gaussiana): omnipresente. Binomial, Poisson, Exponencial. Inferencia: intervalos de confianza, pruebas de hipotesis t-student, ANOVA. Regresion lineal. Teorema de Bayes.
Six Sigma
Control estadistico de calidad
6σ significa 3.4 defectos por millon de oportunidades. Herramientas: DMAIC, cartas de control X-R, Cp y Cpk (capacidad de proceso). La estadistica es el fundamento del control de calidad industrial.

Transformada de Fourier y Senales

Descompone cualquier senal en suma de senoides de diferentes frecuencias. La FFT (Fast Fourier Transform) es el algoritmo mas importante en procesamiento digital de senales.

TRANSFORMADA DE FOURIER
F(ω) = ∫[-∞,∞] f(t)·e^(-jωt) dt
COMPLEJIDAD FFT
O(n log n) vs O(n²) de la DFT directa
APLICACIONES

MP3 usa FFT para comprimir audio. WiFi/5G usa OFDM basado en FFT. MRI medico usa la transformada de Fourier 2D para reconstruir imagenes del interior del cuerpo humano.
